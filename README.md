# 대한민국 전통 시장 가이드 (KorTraMarket) ⛺

전국 각지의 전통시장 정보를 한눈에 확인하고, 실시간 날씨와 로컬 특산품 정보까지 제공하는 반응형 웹 서비스입니다. 

## 📌 주요 기능
- **전국 시장 검색 및 필터링:** 텍스트 검색 및 지역별 카테고리 필터링을 통해 원하는 시장을 빠르게 찾을 수 있습니다.
- **실시간 날씨 정보:** 각 시장의 위치 데이터를 기반으로 실시간 날씨 정보를 시장 카드에 함께 표시하여 방문 전 날씨를 미리 확인할 수 있습니다.
- **로컬 특산품 및 주변 관광지:** 해당 지역에 따른 추천 관광지와 먹거리(특산품) 정보를 제공하여 여행의 즐거움을 더해줍니다.
- **즐겨찾기 및 최근 본 시장:** 사용자가 관심 있는 시장을 하트(❤️)로 즐겨찾기하고, 최근 확인한 시장 목록을 따로 모아보는 편의 기능을 제공합니다.
- **내 주변 시장 찾기 (GPS 모드):** 사용자 위치를 기반으로 반경 20km 이내의 가장 가까운 시장들을 찾아 거리를 표시해 줍니다.

## 🛠️ 사용 기술
- **Frontend:** HTML5, Vanilla JavaScript, Vanilla CSS (모바일 최적화 반응형 웹)
- **Data:** 공공 API 활용 (전국 전통시장 데이터 및 기상 정보)

## 💡 개발자 노트
전국의 매력적인 전통시장들을 더 많은 분들이 편하게 둘러보고 찾아가실 수 있도록 기획한 프로젝트입니다. 모바일 환경에서도 누구나 쉽고 가볍게 사용할 수 있도록 심플하고 직관적인 UI를 구현하는 데 집중했습니다.
