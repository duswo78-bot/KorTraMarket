# 전국 전통시장 찾기 (KorTraMarket) ⛺

사용자 주변 20km 이내의 전통시장을 찾아주고, 전국 각지의 전통시장 정보와 실시간 날씨, 로컬 특산품 정보를 제공하는 반응형 웹 서비스입니다. 

## 📌 주요 기능
- **내 주변 시장 찾기 (GPS 모드):** 하버사인(Haversine) 공식을 활용해 사용자 위치 반경 20km 이내의 가장 가까운 시장을 찾아 거리를 표시해 줍니다.
- **전국 시장 검색 및 필터링:** 텍스트 검색, 지역별 필터링, 그리고 최근 본 시장 및 즐겨찾기 필터 기능을 제공합니다.
- **실시간 날씨 정보:** 각 시장의 위치 데이터를 기반으로 실시간 날씨 이모티콘을 시장 카드에 함께 표시합니다.
- **로컬 특산품 정보:** 지역에 따른 추천 관광지 및 먹거리(특산품) 정보를 카드 확장(아코디언) UI로 보기 쉽게 제공합니다.
- **즐겨찾기 및 캐시 최적화:** 로컬 스토리지(Local Storage)를 활용하여 즐겨찾기와 최근 본 목록을 저장하고, 날씨 API 호출 횟수를 절약하는 캐싱 로직이 적용되어 있습니다.

## 🛠️ 사용 기술 및 API
- **Frontend:** HTML5, Vanilla JavaScript, Vanilla CSS (별도 프레임워크 없음)
- **전통시장 데이터 API:** VWorld 데이터 API 2.0 (LT_P_TRADSIJANG)
- **날씨 데이터 API:** Open-Meteo 무료 API

## 🚀 빠른 시작 가이드 (로컬 실행)
이 프로젝트는 서버 구축 없이도 프론트엔드 코드만으로 동작합니다.
1. 이 레포지토리를 클론(Clone)하거나 Zip으로 다운로드합니다.
2. `KorTraMarket.html` 파일을 브라우저에서 실행합니다.
   * 참고: GPS (내 위치 찾기) 기능은 브라우저 보안 정책 상 **`https://` 환경 또는 `localhost`에서만 작동**합니다.
   * 가장 쉽게 테스트하시려면 이 저장소를 `GitHub Pages`로 배포(Settings -> Pages)하여 사용하시는 것을 권장합니다.

## 💡 개발자 노트
- VWorld API 키는 무분별한 봇 스크래핑을 방지하기 위해 간단한 난독화 처리가 되어 있습니다. (자신의 키로 교체하려면 소스코드 내 `_k` 변수를 수정하세요.)
- 날씨 API는 키가 필요 없는 무료 버전을 사용하므로 자유롭게 활용할 수 있습니다.
- 반응형 웹(Responsive Web)으로 디자인되어 모바일 환경에서도 최적의 UI를 제공합니다.
